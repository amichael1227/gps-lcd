If you are using...

Python 3.7:
Move the two files in this directory to: /usr/lib/python3.7

Python 2.7
Move the two files in this directory to: /usr/lib/python2.7

If you do not move them to these locations, and you want to use the LCD screen with I2C, you MUST include these two files in the same directory as the script, otherwise, it will not run!




The most up-to-date versions of "i2c_lib.py" and "lcddriver.py" can be found at https://github.com/the-raspberry-pi-guy/lcd, which is where they were originally from.
